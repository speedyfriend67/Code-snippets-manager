# Code Snippet Manager

A web-based Code Snippet Manager to organize and categorize your programming snippets with basic syntax highlighting.

## Features

- Add code snippets with associated programming languages.
- Display snippets with syntax highlighting.
- Simple and responsive design.

## Getting Started

1. Clone the repository or download the ZIP file.

   ```bash
   git clone https://github.com/your-username/code-snippet-manager.git
   ```

2. Open `index.html` in a web browser.

3. Add new code snippets using the provided form.

4. View and organize your snippets.

## Usage

- **Add Snippet:** Enter the programming language and your code in the provided form and click "Add Snippet."

- **View Snippets:** All added snippets will be displayed with syntax highlighting.

- **Clear Form:** To add a new snippet, clear the form by refreshing the page or clicking the "Clear Form" button.

## Technologies Used

- HTML
- CSS
- JavaScript
- Basic syntax highlighting using preformatted blocks.
